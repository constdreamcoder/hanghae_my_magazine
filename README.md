## 🚀 미처 완성하지 못한 기능

🤔 회원가입 페이지

- 이메일 형식 체크

🤔 로그인 페이지

- 이메일, 패스워드 미기입 시 로그인 버튼 활성화 막을 것

🤔 메인 페이지(게시글 목록 페이지)

- 게시글 하나를 클릭 시, 게시글 상세 페이지로 이동

🤔 글 작성 페이지

- 텍스트, 이미지 중 입력 안된게 있다면 게시글 작성 버튼 비활성화
